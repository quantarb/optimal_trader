# Portfolio Optimization Research Report

## 1. Experiment setup

- This report is generated by `run_portfolio_optimization_research`.
- The checked-in version is a scaffold so the report path exists in-repo; running the command will overwrite it with current experiment results.
- Objective: compare existing equal-weight cross-sectional quantile portfolios against constrained mean-variance portfolio construction on the same signals.

## 2. Signals tested

- Baseline momentum ranking score.
- Characteristics-factor model predictions.
- Optional external prediction artifacts passed to the management command.

## 3. Optimizer specification

- Objective: expected return minus risk-aversion-scaled variance minus turnover penalty.
- Phase 1 risk models: stabilized sample covariance and factor covariance.
- Phase 1 constraints: gross exposure cap, net exposure target, max single-name weight, with optional sector neutrality.

## 4. Results

- Run `python manage.py run_portfolio_optimization_research ...` to populate the performance comparison, turnover comparison, drawdown comparison, and observations sections with live results from the local research stack.
